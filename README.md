# Controle Financeiro

A personal and group finance management application built with the PERN stack (PostgreSQL, Express, React, Node.js) and powered by Google Gemini AI for financial insights.

## Features

- **Multi-Tenancy (Groups):** Manage finances for yourself, your family, or a project in separate groups.
- **Dashboard:** Interactive charts and summaries of your financial health.
- **Financial Tracking:**
    - **Bills:** Track recurring fixed expenses.
    - **Incomes:** Record your revenue sources.
    - **Investments:** Track assets and simulate yields.
    - **Variable Expenses:** Manage one-off or random spendings.
- **AI Advisor:** Get personalized financial advice and strategies generated by Google Gemini based on your actual data.

## Quick Start (Docker)

1.  **Clone the repository.**
2.  **Create a `.env` file** (see [Setup](docs/setup_installation.md) for template).
3.  **Run with Docker Compose:**
    ```bash
    docker-compose up --build -d
    ```
4.  **Access:**
    - Frontend: `http://localhost:5173`
    - Backend: `http://localhost:3000`

## Development

- **Frontend:** React 19 + TypeScript + Vite (`src/`)
- **Backend:** Node.js + Express (`server/`)
- **Database:** PostgreSQL (`init-scripts/`)
